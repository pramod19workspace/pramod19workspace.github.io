# 01 / about

cat about.md

Hey, I'm Pramod — currently transitioning into a DevOps Engineer role and documenting every step of the journey publicly.

I believe the best way to learn DevOps isn't by reading about it — it's by breaking things in a terminal and figuring out how to fix them. So that's what I'm doing, one tool at a time.

This site grows with me. Every skill listed here is one I've actually touched, tested, and used — no padding, no buzzwords I can't back up. When I learn the next thing, you'll see it appear here. When I don't know something yet, you won't find it pretending to be there.

If you're a recruiter, hiring manager, or fellow learner — welcome. Pull up a chair, grep around. Let's talk.
